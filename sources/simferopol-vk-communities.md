# Реестр публичных VK-сообществ Симферополя

Назначение файла: рабочий реестр публичных сообществ ВКонтакте, которые могут использоваться Агентом Кинодвижа для безопасного OSINT-мониторинга по Симферополю.

> Статус на момент создания: **каркас реестра создан, автоматическое заполнение не завершено**.
>
> Причина: доступный веб-поиск и прямое открытие страниц `vk.com` не дали надёжного способа подтвердить список сообществ. Для полного и проверяемого сбора нужен один из вариантов: официальный VK API access token, выгрузка результатов поиска VK, либо список стартовых публичных групп от пользователя.

## Разрешённый режим работы

Агент может собирать и анализировать только:

- публичные сообщества и публичные посты;
- официальные страницы организаций, ведомств, СМИ и городских сервисов;
- открытые фото, видео, ссылки и текст публикаций;
- дату/время публикации, ссылку на пост, название сообщества и тематику.

Запрещено добавлять в реестр:

- закрытые группы;
- приватные профили;
- персональные страницы жителей;
- токены, cookie, пароли, экспорт приватных сообщений;
- источники, предназначенные для пробива, деанонимизации или сбора личных данных.

## Поисковые запросы для пополнения

Использовать только для поиска публичных сообществ:

```text
Симферополь новости ВКонтакте
Симферополь происшествия ВКонтакте
Симферополь пожар ВКонтакте
Симферополь ДТП ВКонтакте
Симферополь афиша ВКонтакте
Симферополь транспорт ВКонтакте
Симферополь ЖКХ ВКонтакте
Киевский район Симферополь ВКонтакте
МЧС Симферополь ВКонтакте
Администрация Симферополь ВКонтакте
```

## Критерии включения сообщества

Сообщество можно добавить в основной список, если выполнены условия:

1. страница доступна публично;
2. название/описание явно связано с Симферополем или Крымом;
3. публикации имеют отношение к городским событиям, происшествиям, транспорту, афише, службам или новостям;
4. источник не требует обхода доступа;
5. не собираются персональные данные участников или авторов комментариев.

## Критерии исключения

Не включать:

- страницы частных лиц;
- чаты знакомств, купли/продажи с личными телефонами как основным содержанием;
- сомнительные источники без публичного описания;
- сообщества, где основная цель — слив, травля, доксинг или публикация персональных данных.

## Как пополнять

1. Найти публичное сообщество.
2. Открыть страницу и проверить, что она публичная.
3. Зафиксировать название и URL.
4. Определить тематику.
5. Внести в таблицу.
6. Для каждого источника сделать первичную оценку надёжности.
7. Не копировать личные данные пользователей и не сохранять комментарии без необходимости.
