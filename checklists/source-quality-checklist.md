# Чек-лист качества VK-источника

## Базовая проверка

- [ ] Источник публичный.
- [ ] Посты доступны без обхода приватности.
- [ ] У источника понятное название и тематика.
- [ ] Есть история публикаций.
- [ ] Есть ссылки на первоисточники или официальные сообщения.

## Надёжность

- [ ] Источник регулярно исправляет ошибки.
- [ ] Не публикует заведомые слухи как факты.
- [ ] Разделяет мнение и факт.
- [ ] Фото/видео соответствуют тексту публикации.
- [ ] Важные сообщения подтверждаются другими источниками.

## Риски

- [ ] Нет регулярной публикации персональных данных частных лиц.
- [ ] Нет призывов к травле.
- [ ] Нет закрытых/слитых данных.
- [ ] Нет опасных инструкций.

## Оценка

- Высокое доверие: официальный источник или устойчиво точное СМИ.
- Среднее доверие: городской паблик с регулярными подтверждениями.
- Низкое доверие: анонимные сообщения, слухи, непроверенные репосты.
