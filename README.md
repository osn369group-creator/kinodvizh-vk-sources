# Кинодвиж VK Sources

Репозиторий для реестра публичных источников ВКонтакте и смежных открытых каналов, которые агент **Кинодвиж** может использовать для безопасных OSINT-сводок.

## Что хранить здесь

- Списки публичных VK-групп и страниц.
- Категории источников: СМИ, ведомства, городские паблики, тематические сообщества.
- Ключевые слова для мониторинга.
- Оценку доверия к источникам.
- Примечания о рисках и ограничениях.

## Что не хранить

- Персональные данные частных лиц.
- Данные из закрытых групп или приватных профилей.
- Токены, cookie, пароли, дампы.
- Инструкции по обходу ограничений платформ.

## Файлы

- `source-registry-template.md` — шаблон реестра источников.
- `keywords/event-keywords.md` — ключевые слова для поиска событий.
- `checklists/source-quality-checklist.md` — оценка качества источника.
